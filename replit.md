# Kason Khoo Real Estate Website

## Overview
Static website for Kason Khoo, a real estate investment consultant. The site showcases new property projects, second-hand sales, rental services, and loan planning information.

## Project Structure
- `index.html` - Main landing page
- `Project.html` - Project listings page
- `Loan.html` - Loan information page
- `Video.html` - Video content page
- `css/` - Stylesheets (Bootstrap, FontAwesome, custom styles)
- `js/` - JavaScript files (jQuery, Bootstrap, custom scripts)
- `images/` - Image assets
- `webfonts/` - Font files for FontAwesome

## Technology Stack
- HTML5
- CSS3 (Bootstrap framework)
- JavaScript (jQuery)
- FontAwesome icons

## Development
This is a static website served via Python's HTTP server on port 5000.

## Deployment
Configured for static deployment - serves files directly from the root directory.
